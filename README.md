# Atomic Counters using Aerospike-counters (WORK IN PROGRESS)
How to create atomic counters using [Aerospike](www.aerospike.com%)

# Local aerospike datbase using docker
```bash
sudo docker-compose up
```
# Run Dotnet example
```bash
dotnet build
dotnet run
```

# Run Java example
```bash
mvn build
mvn exec:java
```
